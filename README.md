# Sistema de Fila de Clientes em Java

Este projeto implementa um sistema básico de gerenciamento de fila de clientes utilizando a linguagem **Java**.  
O programa permite adicionar clientes, atender o próximo da fila e listar todos os clientes que estão aguardando atendimento, através de um menu interativo no console.

## 📌 Funcionalidades
- **Adicionar cliente**: Insere um novo cliente na fila (até o limite definido).
- **Atender cliente**: Remove o primeiro cliente da fila e mostra quem foi atendido.
- **Listar clientes**: Exibe todos os clientes que estão aguardando atendimento.
- **Menu interativo**: Via console para escolher as opções.

## 🛠️ Tecnologias utilizadas
- Linguagem: **Java**
- Entrada de dados: **Scanner**
- Estrutura de dados: **Array**
